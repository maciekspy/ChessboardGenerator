# Chessboard Generator

A simple, customizable online tool to generate chessboard images.

## Features

-   **Customizable Colors**: Choose colors for light squares, dark squares, border, and coordinates.
-   **Adjustable Dimensions**: Set square size, border width, and font size.
-   **Frames**: Add optional inner and outer frames with custom widths and colors.
-   **Presets**: Quickly switch between styles like Classic, Wooden, Ocean Blue, and more.
-   **Export Options**:
    -   Download as **SVG** (Scalable Vector Graphics).
    -   Download as **PNG** (High-quality raster image).
    -   **Print** directly from the browser (optimized print layout).
-   **Sharing**: Share your specific configuration via URL.
